# 📁 Структура проекту

## Основна структура

```
Energy-main/
│
├── 📄 index.html                    # Головна HTML сторінка
│
├── 📁 css/                          # Стилі
│   └── messenger-buttons.css        # Стилі для кнопок месенджерів
│
├── 📁 js/                           # JavaScript файли
│   ├── config.js                    # ⚙️ Конфігурація контактів (ОБОВ'ЯЗКОВО ОНОВИТИ!)
│   └── messenger-buttons.js         # Функціонал кнопок месенджерів
│
├── 📁 assets/                       # Ресурси
│   └── images/                      # Зображення (опціонально)
│
├── 📄 *.jpg                         # Зображення продуктів та робіт
├── 📄 *.mp4                         # Відео файли
│
├── 📄 robots.txt                    # SEO файл
├── 📄 .gitignore                    # Git ignore
├── 📄 README.md                     # Основна документація
├── 📄 SETUP.md                      # Інструкція з налаштування
└── 📄 STRUCTURE.md                  # Цей файл
```

## Опис файлів

### HTML
- **index.html** - Головна сторінка сайту з усім контентом

### CSS
- **messenger-buttons.css** - Стилі для floating кнопок месенджерів
  - Градієнтні кольори для кожного месенджера
  - Hover ефекти та анімації
  - Адаптивність для мобільних
  - Tooltip підказки

### JavaScript
- **config.js** - Конфігураційний файл з контактами
  - ⚠️ **ВАЖЛИВО:** Оновіть посилання на ваші контакти!
  
- **messenger-buttons.js** - Функціонал кнопок
  - Автоматичне створення кнопок
  - Відстеження кліків
  - Адаптивна поведінка при скролі

## Порядок завантаження файлів

1. HTML завантажується першим
2. CSS файли завантажуються в `<head>`
3. JavaScript файли завантажуються перед `</body>`
4. Порядок JS файлів:
   - `config.js` (спочатку - конфігурація)
   - `messenger-buttons.js` (потім - використовує конфігурацію)

## Налаштування

### Обов'язкові кроки:

1. ✅ Відкрийте `js/config.js`
2. ✅ Оновіть посилання на ваші контакти
3. ✅ Збережіть файл
4. ✅ Відкрийте `index.html` у браузері

### Детальні інструкції:

Дивіться файл `SETUP.md` для детальних інструкцій з налаштування.

## Примітки

- Всі зображення можуть залишатися в корені проекту
- CSS та JS файли мають бути в окремих папках для кращої організації
- Конфігураційний файл дозволяє легко змінювати контакти без редагування основного коду
